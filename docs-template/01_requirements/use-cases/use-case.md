# ユースケース設計書
本ファイルはユースケースごとに作成し、ユースケースID（例: UC-001.md）をファイル名とする。

## ユースケースID
xxxx

## ユースケース名
例: 資産登録

## 概要
本ユースケースで実現する業務内容を簡潔に記載する。

## アクター
- 管理者

## 前提条件
- 管理者がログインしている

## 基本フロー
1. 管理者が資産情報を入力する
2. システムが入力内容を検証する
3. システムが資産情報を登録する
4. システムが登録完了メッセージを表示する

## 代替フロー
- 入力値不正時はエラーメッセージを表示

## 事後条件
- 資産情報がデータベースに保存される
