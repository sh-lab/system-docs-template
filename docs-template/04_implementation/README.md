# 実装支援情報（Implementation）

本ディレクトリは、設計ドキュメント（01_requirements / 02_architecture / 03_designs）
を前提として、実装を行う際に参照する情報を配置するための領域である。

## 本ディレクトリの位置づけ
- 本ディレクトリに記載する内容は、設計そのものを定義するものではない。
- 設計ドキュメントをどのように実装へ落とし込むかを補助することを目的とする。
- 実装に関する具体的な規約・サンプル・Copilot への指示内容は、
  各プロジェクトまたはサンプル側で定義する。

## 補足
- 本テンプレートでは、実装技術やフレームワークを前提としない。
- 実装に関する詳細なルールが必要な場合は、
  派生プロジェクト側で本ディレクトリを拡張することを想定している。
