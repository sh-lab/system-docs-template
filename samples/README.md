# Samples

本ディレクトリは、`system-docs-template` を **どのように使うか** を示すための  
**学習用サンプルリポジトリへのリンク集**です。

ここには設計ドキュメントやソースコードそのものは置きません。  
各サンプルは **単体でチェックアウト可能な独立したリポジトリ**として提供されます。

---

## Available Samples

### HogeFugaPiyo 管理会社 備品貸出システム

- 概要  
  架空の企業 *HogeFugaPiyo 管理会社* における  
  **備品貸出管理システム**を題材にした学習用サンプルです。

  什器・プロジェクタ等の備品を対象に、  
  利用者による貸出申請、利用中管理、返却といった  
  典型的な業務システムの流れを扱います。

- 特徴  
  - `system-docs-template` の各テンプレートを **実際に埋めた設計ドキュメント**付き  
  - 要件定義 → アーキテクチャ → 設計 → **実装（ソースコード）** までを一貫して確認可能  
  - Application / Command / Query Service 分離を **実コードレベルで学習可能**  
  - 業務システムを想定し、複雑すぎない **最小構成**

- Repository  
  - https://github.com/sh-lab/system-docs-sample-equipment-lending

---

## How to Use Samples

1. 上記サンプルリポジトリを **単体で clone / fork** する  
2. サンプル内の `docs/` ディレクトリを読み、  
   `docs-template/` 配下の各テンプレートと対応関係を確認する  
3. 設計ドキュメントと実装コードを照らし合わせて学習する  
4. 自身の学習用や検証用として、ユースケースや設計を追加・改変する
